# Research Agent

**Name:** Research Agent
**Slug:** research-agent
**Version:** 1.0.0

## Purpose

A research assistant that gathers information on topics using web search, synthesizes findings, and produces concise summaries.

## Capabilities

- Web search via configured search tools
- Information synthesis
- Source citation
- Summary generation

## Use Cases

- Market research
- Competitive analysis
- Topic exploration
- Fact checking
